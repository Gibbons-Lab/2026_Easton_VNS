# 2026 Easton VNS microbiome analysis

This repository contains R analysis notebooks and helper functions for the manuscript: Non-invasive Vagal Nerve Stimulation as a Potential Treatment for Repetitive Blast.

## Repository purpose

This is a documentation-first manuscript analysis repository. The code can reproduce the manuscript analyses for users with access to the controlled metadata and regenerated MetaPhlAn outputs.

## Data availability

- Raw sequencing reads: NCBI SRA accession PRJNA1490884.
- Metadata: available upon request.
